# BlindHub 🦯

**BlindHub** هو تطبيق مجتمعي حصري للمكفوفين، مصمم ليكون تجربة صوتية 100% باللغة العربية.

## المميزات الأساسية (الخدمات العشر):
1.  **المنتديات الصوتية**: غرف نقاش صوتية فقط.
2.  **وصف الكائنات بالذكاء الاصطناعي**: وصف فوري عبر الكاميرا.
3.  **الملاحة عبر GPS**: توجيهات صوتية خطوة بخطوة.
4.  **نداء الاستغاثة (SOS)**: طلب مساعدة بلمسة واحدة.
5.  **واجهة قارئ برايل**: دعم الشاشات الخارجية.
6.  **المركز التعليمي**: دورات صوتية متخصصة.
7.  **شبكة الإرشاد**: تواصل مع خبراء وموجهين.
8.  **دردشة صوتية إلى نص**: تواصل سهل وسريع.
9.  **لوحة الوظائف**: فرص عمل متاحة للمكفوفين.
10. **التحكم الصوتي الكامل**: تنقل بدون استخدام اليدين.

## التقنيات المستخدمة:
- **اللغة**: Kotlin
- **النظام**: Android (Target SDK 31)
- **الواجهة**: Voice-Driven UI (No Visual Clutter)
- **الأمان**: Blind-Only Voice Authentication
